Sending configuration to Pocket Miku

When this application is executed, it will change the default behavior of Pocket Miku and it will overwrite all changes that were made by user using a application or SysEx messages.
By using the initialize command (hold VOL UP + VOL DOWN + SHIFT for 5 seconds), you can always return to the factory default state.
When using this application to change the configuration of Pocket Miku, please uninstall maybe previously installed Config first.
It is recommended to execute the initialize command (hold VOL UP + VOL DOWN + SHIFT for 5 seconds).
(When you execute the initialize command, the contents of the lyrics memorized in the memory by the application 01 etc are also reset).

Please pay attention to these points when using this application.
Please use at your own risk.
